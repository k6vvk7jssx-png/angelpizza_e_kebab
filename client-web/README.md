# Angels Livorno Client Website

This is the customer-facing food ordering website for **Angels Pizza & Kebab** in Livorno, built with Next.js and Supabase.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment variables in `.env.local`:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=https://cavxvkwixbxbdvaasxpa.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
3. Run local development server:
   ```bash
   npm run dev
   ```

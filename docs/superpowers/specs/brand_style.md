# Angels Livorno - Brand Style Guide

This document defines the visual identity and design system for the **Angels Livorno Digital Ecosystem**, derived from the restaurant's printed menu/flyer.

## 1. Color Palette

The brand uses a warm, high-energy palette characteristic of fast-food and pizzeria dining:

| Color | Hex Code | Role | Description |
| :--- | :--- | :--- | :--- |
| **Warm Orange** | `#EA580C` (or HSL 20, 92%, 50%) | **Primary Brand** | Represents the main flyer background, energy, and appetite. |
| **Bright Gold/Yellow** | `#FACC15` (or HSL 48, 96%, 53%) | **Secondary Accent** | Used for category headers, badges, and highlights. |
| **Clean White** | `#FFFFFF` | **Backgrounds / Contrasts** | Used for product cards, readability panels, and the main logo text. |
| **Charcoal Black** | `#1C1917` (or HSL 24, 10%, 10%) | **Typography / Borders** | Used for card borders, text, and structure. |
| **Deep Red** | `#991B1B` (or HSL 0, 72%, 35%) | **Alerts / Special Titles** | Used for special offers (e.g. "Grande Inaugurazione", "Pizza Gratis"). |

## 2. Typography

We use two primary typefaces to mirror the flyer's look:

1. **Header Font**: A bold, geometric, modern sans-serif (e.g., **Outfit**, **Inter**, or **Montserrat** in black weights) to match the heavy futuristic logo font "ANGELS".
2. **Body Font**: A clean, highly readable sans-serif (e.g., **Outfit** or **Roboto**) for descriptions and menu lists.
3. **Accent Font (Script)**: A beautiful handwritten/script font (e.g., **Playball** or **Alex Brush**) for callouts like *"Grande Inaugurazione"* or *"Specialità"*.

## 3. Logo and Iconography

- **Logo Mark**: A chef's hat nested between two stylized sharp wings, centered above the wordmark.
- **Wordmark**: "A N G E L S" in all-caps, futuristic, wide geometric letters.
- **Borders & Framing**: Heavy rounded boxes (`border-radius: 8px` to `12px`), clean double borders in black/white, and high contrast spacing.

## 4. UI Elements & Layout System

- **Category Badges**: Rounded rectangles with yellow backgrounds and black text.
- **Product Cards**: White boxes with solid black borders (`1.5px solid #1C1917`) and a subtle flat shadow effect (neo-brutalism style) rather than soft blur shadows.
- **Menu Items**: Horizontal listings with dotted dividers matching the traditional Italian restaurant menu layout:
  ```text
  MARINERA pomodoro, aglio, origano ............................ € 6,00
  ```
